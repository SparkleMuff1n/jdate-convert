[Installation]
* Download the .xpi file
* Zotero → Tools → Add-ons → gear icon (⚙) → Install Add-on From File…
* Select the .xpi
* Restart Zotero if necessary

[Usage]
1. Select one or more items in your library.
2. Right-click → "Convert Japanese Dates", or Tools → "Convert Japanese Dates (selected items)"

[Notes]
* Use the dropdown in Tools to select a date output format. The default is yyyy.mm.dd.
* A summary dialog will tell you how many items were converted vs. skipped.
* Only the five modern eras are supported: 令和, 平成, 昭和, 大正, 明治.
* Double-width (全角) characters are recognized and automatically converted.
* Items with no recognisable Japanese date in the Date field are silently skipped.

[Examples]
* 2025年3月14日, 令和7年3月14日 → 2025.03.14
* 2025年3月14日, 令和7年3月14日 → 2025.03.14
* 2025年3月 → 2025.03
* 2025年 → 2025
